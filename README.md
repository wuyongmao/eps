# 采购系统eps


### 版本
 - JDK 1.7+
 - Oracle
 - druid
 - Maven 


### 整合:SpringMVC+Mybatis +EasyUI +shiro


### 主要功能
 + mybatis 分页
 + mbg.xml 为逆向创建dao、mapper配置文件。直接运行生成    com.chenyuantech.eps.test.MBGTest 
 + 定时发送邮件
